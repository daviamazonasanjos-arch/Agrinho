# Agrinho 2026
🚀 Funcionalidades
Painel de Regiões Interativo: Apresenta dados específicos sobre as características agrícolas e desafios das cinco regiões brasileiras (Sul, Sudeste, Centro-Oeste, Norte e Nordeste).

Simulador de Impacto (Mini-jogo): Permite ao usuário aplicar ações (como o uso de bioinsumos ou maquinário pesado) e visualizar em tempo real o impacto dessas escolhas nas barras de progresso de Sustentabilidade e Produção.

Quiz Educativo: Um sistema de perguntas com validação instantânea para testar os conhecimentos do usuário sobre práticas agrícolas regenerativas.

Interface Moderna e Responsiva: Design otimizado para computadores e dispositivos móveis, utilizando efeitos visuais como blur (desfoque de fundo) e animações de pulsação de status.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo e das abas de navegação.

CSS3: Estilização personalizada, layout em Grid/Flexbox, efeitos de hover, transições suaves, estilização da barra de rolagem e responsividade para telas menores.

JavaScript (ES6): Manipulação do DOM para alternância de abas, lógica do simulador de pontuação e validação das respostas do quiz.
